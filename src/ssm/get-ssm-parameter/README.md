Get SSM Parameter
=========

## Simple Usage

```
./get-ssm-parameter.py
```

## Command Line Options

```

usage: get-ssm-parameter.py [-h] -n NAME [-r REGION]

Get SSM Parameter

optional arguments:
  -h, --help            show this help message and exit
  -n NAME, --name NAME  The name of the parameter to lookup
  -r REGION, --region REGION
                        The aws region

```
