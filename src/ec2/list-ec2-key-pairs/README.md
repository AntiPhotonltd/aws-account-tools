List EC2 Key Pairs
=========

## Simple Usage

```
./list-ec2-key-pairs.py
```

## Command Line Options

```

usage: list-ec2-key-pairs.py [-h] [-r REGION]

List EC2 Key Pairs

optional arguments:
  -h, --help            show this help message and exit
  -r REGION, --region REGION
                        The aws region

```
