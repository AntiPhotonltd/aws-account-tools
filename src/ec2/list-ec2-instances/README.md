List EC2 Instances
=========

## Simple Usage

```
./list-ec2-instances.py
```

## Command Line Options

```

usage: list-ec2-instances.py [-h] [-a] [-r REGION]

List EC2 Instances

optional arguments:
  -h, --help            show this help message and exit
  -a, --all             Show all instances
  -r REGION, --region REGION
                        The aws region

```
